Last Time I Did - Support

Welcome to the Support Page

Thank you for using Last Time I Did! This app helps you keep track of tasks you perform on a regular basis and reminds you when it's time to do them again.

How to Use the App

Follow these simple steps to get started:

  • Enter a Task: Tap the "+" button to enter a task you want to track (e.g., "Change air filter", "Clean windows").
  • Set a Reminder: For each task, you can set a reminder so you won’t forget when it’s time to do it again.
  • Check Your Last Action: The app will display when you last completed a task so you can easily see if it’s time to do it again.
  • Edit or Delete Entries: If you make a mistake, you can always edit or delete a task by tapping on it.

What You Can Do with the App

This app allows you to:

  • Track recurring tasks like changing filters, cleaning, or maintenance.
  • Set and receive push notifications as reminders.
  • View a list of completed tasks and how long it has been since the last one.
  • Customize your task list and modify or delete tasks as necessary.

FAQs

Q: How do I reset my task reminders?

A: Simply go to your task list, tap the task you want to reset, and adjust the reminder settings.

Q: Can I sync this app with my calendar?

A: At the moment, the app does not support calendar syncing, but we are planning to add this feature in the future!
